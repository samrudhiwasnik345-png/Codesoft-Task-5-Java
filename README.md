# 🎓 Student Management System :-

A simple **Student Management System** built using **Core Java** that allows users to manage student records through a menu-driven console application. 
The project demonstrates the use of **Object-Oriented Programming (OOP)** concepts, **File Handling**, **Collections**, and **Exception Handling**.

---

## 📌 Features

- ➕ Add New Student
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Edit Student Details
- ❌ Remove Student
- 💾 Save Student Records to File
- 📂 Load Student Records Automatically on Startup
- 🖥️ Menu-Driven Console Interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collection
- File Handling (BufferedReader, FileWriter, PrintWriter)
- Exception Handling

## 💾 File Storage

Student records are stored in:

```
students.txt
```

Each record is saved in the following format:

```
RollNumber,Name,Grade
```

Example:

```
101,John,A
102,Emma,B+
103,David,A+
```

The application automatically loads the saved data whenever it starts.
